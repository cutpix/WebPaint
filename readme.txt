WebPaint - многопользовательская векторная рисовалка. 	

1.	Пользователь создает Email аккаунт или логинится через Facebook
2.	После логина, пользователь попадает на главную страницу, на ней находится область для рисования и инструменты для рисования.
3.	Все пользователи, которые онлайн, рисуют в одной области. Изменения каждого пользователя должны мгновенно синхронизироваться между всеми остальными пользователями
4.  Пользователь может отменить до 5 последних своих действий. 
5.	Каждый пользователь может управлять только своими элементами.
6.	Пользователь может сделать share текущей картинки в Facebook

Проект на Visual Studio 2017 Community. Используется база MS SQL Server. Серверная служба asp.net web api, клиетская сторона - html + angularjs. 
Для развертывания в дебаге достаточно запустить проект в VS. База создается автоматически.

Ограничение: если пользователь зарегистрировался с email через facebook, то как обычная регистрация пройдет с ошибкой "email is already taken".

При первой регистрации через facebook происходит ошибка Unauthorized, при повторном логине ошибка пропадает.

Share to facebook: не работает на локальный url.
